## Instantiation 
 
- Inject the domain via the constructor in the shared OtrService variable 
```js 
... 
var otrService = new OtrService({domain: ENV.apiEndpoint}) 
... 
 
## Usage 
 
- The methods generated in the OtrService component are the methodName + "Using" + httpMethod 
- E.g. getCaseUsingGET, confirmBookingUsingPOST, and etc... 
 
```js 
otrService.getCaseUsingGET({caseId: caseId}).then(...) 
``` 
 
## Error Handling 
 
- Sample Error 
 
```js 
otrService.getCaseUsingGET({caseId: caseId}) 
    .then( 
        function(response) { 
         
        }, 
        function(error) { 
            var uiErrorMsg = error.body.error.uiErrorMsg; 
            ... 
        } 
    ); 
```
